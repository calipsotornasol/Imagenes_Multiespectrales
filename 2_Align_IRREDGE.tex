    This code uses the Python Open CV library to align red and rededge
images in the format tif and combines them to produce an aligned RGB
image in the format JPG. In addition, this code transfers the GPS
information from the image in the reference band to the aligned RGB
image.

Extensions asociated to each name file refers to a diferente band as is
detailed below.

\begin{itemize}
\tightlist
\item
  '*\_1.tif' = Blue Band
\item
  '*\_2.tif' = Green Band
\item
  '*\_3.tif' = Red Band
\item
  '*\_4.tif' = Near Infrarred Band
\item
  '*\_5.tif' = Rededge Band
\end{itemize}

The output can be used to compute the NDRE vegetation index.
